# NoteFlux

Сайт заметок с паролем на входе, админ-панелью и сохранением данных.

## Локальный запуск

```bash
python3 app.py
```

Откройте `http://localhost:3000`.

- Пароль сайта по умолчанию: `1234`
- Пароль админ-панели по умолчанию: `admin123`

## Деплой на Vercel

Проект подготовлен под Vercel Python runtime (`vercel.json`).

### Важно про сохранение данных

Для **локального запуска** данные пишутся в `data/store.json`.

На Vercel файловая система не является постоянной между деплоями/инстансами,
поэтому для реального продакшн-хранения заметок лучше подключить внешнюю БД/хранилище
(Vercel KV, PostgreSQL, Supabase и т.д.).
